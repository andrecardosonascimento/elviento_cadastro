**ElViento** é um projeto de backend desenvolvido com foco em boas práticas de arquitetura, organização modular e uso 
eficiente de tecnologias Java modernas. Criado para compor meu portfólio, ele simula uma aplicação RESTful robusta, 
escalável e pronta para produção usando como base um modelo de dominio completo, usando o spring boot para o desenvolvimento, para a base de dados será usado o postgreSQL

Tarefas do projeto

- Criar projeto Spring Boot Java
- Implementar modelo de domínio
- Estruturar camadas lógicas: resource, service, repository
- Configurar banco de dados de teste (H2)
- Povoar o banco de dados usando conexão a base de dados
- CRUD - Create, Retrieve, Update, Delete
- Realizar Tratamento de exceções


Este projeto foi desenvolvido com o propósito de demonstrar domínio técnico em Java e Spring Boot, além de servir como
base para aplicações reais e estudos futuros. Ele está estruturado para facilitar a manutenção, extensibilidade e compreensão 
por outros desenvolvedores.
